# STM32-MICROCONTROLLER-USB-DEVELOPMENT-BOARD
Custom STM32 microcontroller USB development board designed using KiCad. Includes LDO based 3.3V power regulation, external crystal oscillator, decoupling capacitors, USB interface and GPIO expansion headers on a 2-layer PCB with ground plane.

##Features

- STM32 microcontroller based development board
- USB interface for communication and power
- 3.3V LDO voltage regulator
- External crystal oscillator for stable clock
- Decoupling capacitors for power stability
- GPIO expansion headers
- 2-layer PCB with ground plane
